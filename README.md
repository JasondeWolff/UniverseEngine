# UniverseEngine
UniverseEngine is an engine made for space games, which we build an example exploration game with.

## Features
- Spaceship movement
- Interstellar physics
- Terrain generation
- Planet populator
- Planet viewer tool
- USD assets
- OpenGL renderer
- World streaming
